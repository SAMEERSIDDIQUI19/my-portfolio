# Profile Picture Folder

Place your profile picture in this folder with one of these names:
- `profile.jpg`
- `profile.png` 
- `profile.jpeg`
- `profile.gif`
- `profile.webp`

The image will automatically be displayed on your portfolio home page.

## Instructions:
1. Add your profile picture to this folder
2. Name it exactly as one of the supported filenames above
3. The picture will appear automatically on your home page

## Recommended:
- Image size: 300x300 pixels or larger (square format works best)
- File size: Under 1MB for faster loading
- Format: JPG or PNG recommended
